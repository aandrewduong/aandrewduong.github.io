PROJECTS

Healthy on The Hill - [Project Repository]

Healthy on The Hill is a web application designed to improve the health and ease of connection for UCLA students living on the Hill. Built using React, JavaScript, Golang, Git, and MongoDB, the platform enables seamless communication and data exchange through robust RESTful APIs. I collaborated with Ananya Anand, Steven Veld, Gabriel Scharff, Anthony Chui, and Hayleigh Liu, utilizing Git for version control to streamline our workflow and ensure efficient project management. Together, we designed an intuitive user interface and implemented backend services to support real-time interactions. Additionally, I leveraged web scraping techniques to dynamically extract and integrate real-time data from UCLA websites, ensuring students have access to up-to-date and relevant information.

Healthy on The Hill allows students to connect with gym buddies based on fitness goals and levels, create personalized fitness plans with recommended gym equipment, view live food options on the Hill with meal ratings, and check live activity levels of dining halls and gyms. Through this project, we gained valuable experience in full-stack development, team collaboration, and optimizing web applications for a smooth and responsive user experience.

De Anza User Automation Tool - [Project Repository]

De Anza User Automation Tool is a full-stack application that streamlines course enrollment at De Anza and Foothill College by automating class tracking and registration. With a Golang backend handling API requests, authentication, and registration logic, and an Electron-based frontend providing an intuitive interface, the tool simplifies the enrollment process. It uses real-time web scraping and HTTP requests to monitor course availability, automatically enrolling students when seats open. Features include Class Search & Export, Unofficial Transcript Retrieval, Rapid Enrollment, and Watch Mode for real-time alerts and automatic sign-ups. Data persistence and user session management ensure a seamless experience.
